# Eco-Track — Danang Nur Himawan

## Tentang Project
Eco-Track adalah aplikasi web sederhana yang membantu pengguna menghitung jejak karbon harian mereka berdasarkan aktivitas transportasi dan penggunaan energi. Pengguna cukup memasukkan data seperti jarak tempuh kendaraan, durasi penggunaan AC, dan durasi penggunaan laptop, lalu aplikasi akan menghitung total emisi CO₂ yang dihasilkan dalam sehari beserta kategorinya.

## Masalah Iklim yang Ingin Disoroti
Banyak orang tidak menyadari bahwa aktivitas sehari-hari seperti berkendara dan menggunakan perangkat elektronik berkontribusi nyata terhadap emisi karbon. Kurangnya kesadaran ini membuat perubahan perilaku sulit terjadi. Melalui Eco-Track, saya ingin membantu pengguna melihat secara langsung seberapa besar dampak aktivitas mereka terhadap lingkungan, sehingga dapat mendorong kebiasaan yang lebih ramah lingkungan.

## Cara Menjalankan Project
1. Clone atau extract folder project
2. Buka file `public/index.html` langsung di browser 
3. Isi semua input yang tersedia, lalu klik tombol **Hitung**
4. Klik **Reset** untuk mengosongkan semua input
